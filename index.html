My Portfolio

Analysis Report for Fraud Prevention Project

This project performs an exploratory analysis of the "creditcard.csv" dataset to detect patterns of fraudulent activities in financial transactions. The analysis includes class distribution, variable correlation, outlier analysis, and visualizations such as histograms and 3D plots. Conclusions suggest using advanced machine learning techniques to improve fraud detection, given the non-linear characteristics and class imbalance in the data.

Impact of Plastic Waste in Florianópolis and the World: Exploring the Boundaries between Local and Global

This project investigates the improper disposal of plastic waste, focusing on PET bottles, in Florianópolis and compares it with national and global data. Using real data, interactive graphs, and 3D visualizations, we aim to understand the problem's magnitude and propose solutions to reduce environmental impact.

Financial Data Analysis Project

This project involves analyzing historical financial market data to understand trends in opening, closing, high, and low prices, and trading volume. The analysis includes temporal trend visualizations, trading volume examination, and identifying high volatility periods. Interactive 3D plots and statistical summaries provide insights to inform strategic decisions and risk management.

Analysis of Online Consumer Profile and Behavior

Análise Exploratória de Dados (EDA)

Este projeto envolve uma análise abrangente dos dados de seguro de saúde, com o objetivo de entender melhor os clientes e melhorar a estratégia de preços da empresa de seguros.

Análise e modelagem

As etapas incluem leitura dos dados, visualização gráfica, cálculos estatísticos, preparação dos dados para modelagem, treinamento e avaliação do modelo de regressão linear. Os resultados gráficos, testes de hipóteses e coeficientes do modelo, fornecem insights valiosos para a empresa de seguros.

Análise exploratória de dados (EDA) dos comandos do Kubectl

About Me

Ivan Melo - Especialista em Ciência de Dados

Olá! Me chamo Ivan Souza de Melo e sou apaixonado pelo mundo da Ciência de Dados. Como especialista nessa área fascinante, trago um amplo conhecimento em diversas linguagens e ferramentas essenciais para o universo da análise de dados.

Minha jornada profissional é marcada pela expertise em linguagens de programação como Python, R e SQL, que utilizo para extrair insights valiosos dos dados e criar soluções inteligentes. Além disso, domino o ambiente do Power BI para visualização e análise de dados de forma dinâmica e impactante.

Como professor, tenho o privilégio de compartilhar meu conhecimento em desenvolvimento web, desenvolvimento de software e, é claro, Ciência de Dados. Acredito profundamente na importância da educação para capacitar novas gerações de profissionais e impulsionar a inovação em todas as áreas.

A arte da programação é minha paixão e minha fonte de inspiração diária. Porém, é na Ciência de Dados que encontro meu verdadeiro entusiasmo. A possibilidade de explorar, analisar e interpretar dados para gerar impacto positivo é o que me motiva e me impulsiona a buscar sempre mais conhecimento e excelência na minha jornada profissional.

Seja bem-vindo ao meu mundo, onde os dados são a matéria-prima para construir um futuro mais inteligente e promissor. Estou pronto para desbravar novos desafios e contribuir para o avanço da Ciência de Dados em todas as suas dimensões.

Vamos juntos explorar o poder dos dados e transformar informações em insights poderosos!
